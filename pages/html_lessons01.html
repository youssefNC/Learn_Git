حل تكليفات HTML الدروس من الدرس 01 إلى 05

الحلول لمساعدة الاشخاص الذين يجدون صعوبة في حل التكليفات. لكن قبل رؤية الحل حاول البحث ومحاولة ايجاد الحل بنفسك. ايضا شاهد الدروس مرة اخرى من هنا لترسيخ المعلومات في عقلك.

التكليف 01

رابط التكليف على موقع الزيرو هنا 

<!-- HTML = HyperText Markup Language -->

HTML هي اللبنة الأساسية للويب. نستعملها لتحديد هيكل محتوى الويب. تُستخدم التقنيات الأخرى إلى جانب HTML مثل (CSS) لاعطاء مظهر لصفحة الويب. و (JavaScript) لتحديد سلوك او وظيفة صفحة الويب.

التكليف 02

رابط التكليف على موقع الزيرو هنا 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My First</title>
    <meta name="description" content=" This Is Description For My First Page" />
    <!-- css-path = css مسار ملف -->
    <link rel="stylesheet" href="css-path">
    
    
    <style>
         /* CSS code here */
    </style>
    <script>
        // JavaScript code here
    </script>

</head>
<body>
    This Is My First Page
</body>
</html>

الإعلان <!DOCTYPE html> ليس علامة HTML. إنها "معلومات" للمتصفح حول نوع المستند المتوقع.

التكليف 04

رابط التكليف على موقع الزيرو هنا 

<!DOCTYPE html>
<html lang="en">
<head>
    <title>My First</title>
    
    <!--
            meta 1 => Specifies the character encoding for the HTML document | encoding UTF-8 is supported arabic language
            meta 2 => Define a description of your web page
            meta 3 => Define keywords for search engines
            meta 4 => Define the author of a page
            meta 5 => Setting the viewport to make your website look good on all devices
    -->
    <meta charset="UTF-8">
    <meta name="description" content="وصف صفحتك هنا" />
    <meta name="keywords" content="الكلمات المفتاحية هنا مع الفصل بين كل كلمة مفتاحية واخرى بعلامة الفاصلة" />
    <meta name="author" content="اسم المؤلف هنا" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>
<body>
    
</body>
</html>

<meta> هي علامة HTML تحتوي على بيانات وصفية حول صفحة ويب ، أو بشكل أكثر تحديدًا ، واصِفات تخبر محركات البحث بنوع المحتوى الذي تحتويه صفحة الويب والذي يكون مخفيًا عن العرض.

  • charset هي سمة HTML تحدد ترميز الأحرف لمتصفحك لاستخدامه عند عرض محتوى موقع الويب. و القيمة utf-8 هو ترميز أحرف محدد. بعبارة أخرى ، يخبر <meta charset = "utf-8"> المتصفح أن يستخدم ترميز أحرف utf-8 عند ترجمة كود الجهاز إلى نص يمكن للبشر قراءته والعكس بالعكس ليتم عرضه في المتصفح.
  • name هي سمة HTML تستخدم لتعريف الوصف "description" والكلمات المفتاحية "keywords" ومؤلف المستند "author". تستخدم أيضًا "viewport" للتحكم في أبعاد الصفحة وقياسها للأجهزة المختلفة
    • description يمكنك تعيينه لمنشور أو صفحة من موقع الويب الخاص بك. تظهره محركات البحث في نتائج البحث في الغالب عندما تكون العبارة التي تم البحث عنها ضمن الوصف. لذا فإن تحسينه أمر بالغ الأهمية لتحسين محركات البحث على الصفحة.
    • keywords تستخدم لتحديد الكلمات المفتاحية لمحركات البحث. يتم بعد ذلك استخدام الكلمات المفتاحية التي تضعها في سمة محتوى هذا العنصر كعامل ترتيب بواسطة محركات البحث. " لكن Google لم تعد تستخدم الكلمات الرئيسية في ترتيب الويب" لذا لا تستعملها. (فقط احببت ان تكون على دراية بالموضوع)
    • author تستخدم لتعيين اسم الشخص الفعلي (المؤلف) الذي أنشأ الصفحة. عند تعيينه فمن السهل العثور على الشخص المناسب المسؤول عن إنشأ صفحة الويب.
    • viewport تستخدم لتحسين عرض محتوى الويب الخاص بك.عن طريق تعيين العرض والمقياس الأولي لإطار العرض. بدون علامة viewport META ، ستواجه صعوبة كبيرة في ضمان ظهور موقع الويب الخاص بك كما هو عبر العديد من الأجهزة المحمولة المختلفة ، نظرًا لأن معظم الأجهزة المحمولة لها منافذ عرض مختلفة. قد تبدو الخطوط كبيرة جدًا على أحد الأجهزة ، وصغيرة جدًا على جهاز آخر ، حتى عند استخدام وحدات القياس على أساس النسبة المئوية (القياس على أساس النسبة المئوية معلومة خاصة ب CSS لذا من الطبيعي عدم استيعاب الامر الان). مثال لضبط عرض viewport عن طريق استخدام "content="width=device-width
