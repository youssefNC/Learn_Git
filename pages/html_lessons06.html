    <input type="reset" value="Empty Form" >
</form>
  • تحدد السمة method كيفية إرسال بيانات النموذج وهناك ثلاتة طرق لإرسال البيانات وهي get, post, dialog
  • عند ارسال بيانات النموذج باستخدام POST لا تظهر البيانات في عنوان URL
  • السمة action تحدد الصفحة التي يتم إرسال بيانات النموذج اليها.
  • السمة required تحدد أنه يجب ملء حقل الإدخال قبل إرسال النموذج.
  • السمة name تستخدم لتحديد اسم العنصر input
  • تحدد سمة placeholder تلميحًا قصيرًا يصف القيمة المتوقعة لحقل الإدخال.
