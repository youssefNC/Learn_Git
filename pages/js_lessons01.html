حل تكليفات JavaScript الدروس من الدرس 01 إلى 09

الحلول لمساعدة الاشخاص الذين يجدون صعوبة في حل التكليفات. لكن قبل رؤية الحل حاول البحث ومحاولة ايجاد الحل بنفسك. ايضا شاهد الدروس مرة اخرى من هنا لترسيخ المعلومات في عقلك.

التكليف 01

رابط التكليف على موقع الزيرو هنا 

    <!DOCTYPE html>
    <html lang="en">
      <head>
        <meta charset="UTF-8" />
        <meta http-equiv="X-UA-Compatible" content="IE=edge" />
        <title>Learn JavaScript</title>
        <script>
          // Code One
          document.getElementById("el").style.color = "red";
        </script>
        <script>
          // Code Two
          window.onload = function () {
            document.getElementById("el").style.color = "red";
          };
        </script>
      </head>
      <body>
        <h1 id="el">Page Title</h1>
        <script>
          // Code Three
          document.getElementById("el").style.color = "red";
        </script>
      </body>
    </html>
  • code one لن يعمل لأن الصفحة لم يتم تحميلها بعد خاصة العنصر h1
  • code two سوف يعمل لان الحدث onload يقوم بتنفيذ كود JavaScript فورًا بعد تحميل الصفحة
  • code three سوف يعمل لأنه أخر عنصر في ملف ال HTML. أي أن الصفحة قد تم تحميلها. ملاحظة, المتصفح يقوم بتنفيد الكود من الأعلى نحو الأسفل
